# Corobo Bot
Corobot is a python discord bot utility for the Corobo Cult discord server

## Requirements
### Poetry
1. Install poetry
2. Install sqlite-devel 
3. Run `poetry install`
4. `poetry run python main.py`

### No Poetry
Install
1. discord.py 2.7.1
2. pysqlite3 0.6.0
3. sqlparse==0.5.5
4. figure the rest out somehow
